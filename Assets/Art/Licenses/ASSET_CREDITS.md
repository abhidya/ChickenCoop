### How to Add Production Assets

1. Download assets from Kenney.nl or OpenGameArt.org
2. Replace placeholder sprites in `Assets/Art/Sprites/`
3. Update this file with proper attribution if CC-BY license
4. Configure import settings:
   - Texture Type: Sprite (2D and UI)
   - Pixels Per Unit: Match your game scale (typically 32-100)
   - Filter Mode: Point (for pixel art) or Bilinear (for smooth art)
